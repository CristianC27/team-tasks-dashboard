Backend .NET 8 API

Descripción:

API REST para la gestión de proyectos y tareas.  
Se conecta a PostgreSQL 17 para persistencia de datos y provee endpoints para proyectos, tareas y reportes de resumen.  
Separa la lógica del backend del frontend (SPA Angular), facilitando despliegue y mantenimiento.

Requisitos:
- Node.js >= 16
- npm o yarn
- PostgreSQL 17
- Git
